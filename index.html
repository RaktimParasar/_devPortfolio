Projects that I've worked on

Blog App

Reattractive

This project is all about building content and comments web applications which are common across a large number of websites and applications, from news sites to blogs like Hacker News and Reddit. I have used ReactJs for building the application UI and ReduxJs to manage the application state. The features in this app are, Users will be able to post content to predefined categories, comment on their posts and other users' posts, and vote on posts and comments. Users will also be able to edit and delete posts and comments.

Social Network App

Socialfordev

A full-stack social network application with the idea of connecting developers, sharing a post, and helping each other. Users can create profiles, log in to their accounts, check other dev profiles. Its has all the CRUD application abilities to create a post, read others post, edit and update their post and delete a post. Also, users can make comments, upvote, and downvote on other's posts. The UI is built on ReactJs and the application state is managed by ReduxJs.

Tracking App

Bookshelf

This is a book tracking application that allows you to select and categorize books you have read, are currently reading, or want to read. All the interactive stuff on the app is built on reactJS. The app communicates with an API to get all the book data. Also, it has a search feature where you can give a text input, and the books that match that query are displayed on the page, along with a control that lets you add the book to your library.

Booking App

Beach-Resort

A frontend application integrated with a content management service called Contentful which communicates with an API to work seamlessly. The application allows users to find a perfect room according to their needs by filtering for different room types, price, the number of rooms, also for food and pets. Every selected room is provided with all the details of accommodation so that the user can get a good idea of the room before booking. All data of this application is store in Contentful, React's ContextApi is used to manage the application state. The UI is made on React and it is a Responsive design.

Portfolio Website

DevPortfolio

This is my website/portfolio where I can showcase all the skills and the projects I have done so far. It is a practice project to upskill my Html, CSS, and JavaScript knowledge to build this website. Also used Sass for styling, vanilla JS for some interactive sections, and added animation with GASP JavaScript animation library. The website is completely responsive for all types of devices.
